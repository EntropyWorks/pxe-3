Supported Adapters

Intel® Ethernet iSCSI Boot is supported on all Intel PCI-E Server Adapters and LOMs. A list of supported Intel Server Adapters can be found here.

The iSCSI Initiator is supported on 82599 and X540-based adapters, such as the Intel® Ethernet Server Adapter X520.
