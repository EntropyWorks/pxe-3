Intel® Ethernet iSCSI Boot Installation and Setup

This section provides how to install and setup Intel® Ethernet iSCSI Boot components. The first step after hardware preparation is to program Intel Ethernet iSCSI Boot firmware on an iSCSI Boot-capable adapter. See the Installing Firmware section below.

The default iSCSI Boot configuration uses DHCP. If using Static IP, it is necessary to set up the firmware for static IP configurations. See the section Firmware Setup.

Depending on the target OS to be booted from the iSCSI target disk volume, follow the detailed instructions in Microsoft* Windows Setup or Linux Setup.

Installation Overview for Intel® Ethernet iSCSI Boot

For information on System Requirements see System Requirements.

  1. Install Intel® Ethernet iSCSI Boot-capable adapters in the system. For a list of supported adapters, see Supported Adapters.
  2. Boot via DOS.
  3. Install the latest Intel Ethernet iSCSI Boot firmware using BootUtil DOS utility.
    Command "BOOTUTIL -UP=iSCSI -ALL" programs the firmware on flash ROM of all iSCSI Remote Boot-capable adapters in the system. See BootUtility section for details.
  4. Connect the adapter(s) by cable in the iSCSI initiator system to a network with the iSCSI target disk.
  5. If employing Static IP follow the Firmware Setup instructions to configure the Intel iSCSI Boot firmware to boot from a pre-configured iSCSI target
  6. To setup the initiator system to boot from a remote iSCSI disk using Microsoft* Windows*, follow the steps in Windows Setup.
  7. To setup the initiator system to boot from a remote iSCSI disk using Linux, follow the steps in Linux Setup.
  8. After following the above steps if you are having problems, refer to Troubleshooting.

Installing iSCSI Boot Firmware

In order to enable Intel® Ethenet iSCSI Boot, the firmware should be loaded as Option ROM while booting the BIOS in the system. "Intel® iSCSI Boot" followed by the firmware version number is displayed, if the firmware is installed correctly.

If you are installing or upgrading Intel Ethernet iSCSI Boot firmware on Intel PRO/1000 PT/PF Single/Dual/Quad Port Server Adapter, BootUtil is a DOS utility software used to install the firmware.  BootUtil is not intended as a utility for LAN-On-Motherboard (LOM) implementations.  If you are using an Intel Ethernet iSCSI Boot-capable OEM server system, please refer to instructions provided from the OEM to enable Intel Ethernet iSCSI Boot.
